# Currents 2021

Official Frontend Repo for Currents 2021

Portal Link: [www.currents.nitt.edu](https://www.currents.nitt.edu)

Built using React. 

### Installation
- Clone/Download this repository
- Install dependencies using 
```
    $ npm install
```
### How to Run
- For development
```
    $ npm start
```
- For production
```
    $ npm run build 
```
